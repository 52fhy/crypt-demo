# crypt-demo

## AES

AES-ECB 测试：
```
加密密钥：1234567812345678
加密向量：1234567812345678
加密填充算法：Pkcs7
加密分组模式：ECB
加密密文：123456
加密结果：mdSm0RmB+xAKrTah3DG31A==
备注：ECB模式下加密向量iv没有用到。
```

AES-CBC 测试：
```
加密密钥：1234567812345678
加密向量：1234567812345678
加密填充算法：Pkcs7
加密分组模式：CBC
加密密文：123456
加密结果：2eDiseYiSX62qk/WS/ZDmg==
```
